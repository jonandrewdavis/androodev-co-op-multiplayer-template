# androo-dev-co-op-multiplayer

A complete template and tutorial from [https://www.youtube.com/@AndrooDev](https://www.youtube.com/@AndrooDev). Uses client authority to make it easy to start a PVE or Co-Op multiplayer game from scratch.

## Local Development:

- Clone this project
- Import in Godot 4.6
- Select Debug
  - Select Customize Run Instances
  - Set 3 instances
  - Add feature `server` to the first instance
- Play
- You should see the server start and 2 clients
- Join on client
- Join on client

## Controls

- WASD to Move
- Mouse to look
- Left Click to shoot ball
- Tab to open menu
  - Leave Game
  - Rejoin
